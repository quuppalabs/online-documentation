Q17 Locator Parts

The Quuppa Q17 contains the following parts:

  1. Quuppa Q17 main unit
  2. Quuppa Q17 mounting bracket
  3. Quuppa Q17 connector cover

Q17 Main Unit

  1. Indicator light
  2. Bracket mounting channel
  3. Type label
  4. Ethernet connector
  5. Micro USB connector
  6. Reset button

Q17 Mounting Bracket

  1. Orientation indicator and orientation guideline
  2. Installation guide
  3. VESA standard 75mm x 50mm and 50mm x 50mm
  4. Cable passage
  5. Locking mechanism

Q17 Connector Cover

  1. Cable channel
  2. Locking mechanics
  3. Optional cable passages for thicker cable
