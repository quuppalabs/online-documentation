Create 3D Tag Group

Tag groups are used to group similar type of tags together. This section will walk you through adding a new 3D tag group for your project.

  1. Open the project in the QSP.
  2. In the object tree on the left, find Tag Groups and right click on it.
  3. Select Add Tag Group and give the group a name.
  4. Enable Position in 3D if available option in tag group settings.

  5. Read more about tag group settings in the Tag Groups section.
