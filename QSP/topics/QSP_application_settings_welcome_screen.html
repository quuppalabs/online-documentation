Edit Welcome Screen Settings

By default, the QSP will show a welcome screen when it starts up. The welcome screen gives you quick access to the new project wizard, existing projects stored locally or in the Quuppa Customer Portal and shows recently opened projects.

The QSP application settings enable you to turn this feature off, in which case the QSP will automatically open to the QSP main view on start up, rather than showing you the welcome screen first.
  1. Open the QSP.
  2. In the menu bar at the top, select the Settings menu and select Application Settings. The Application Settings window will open.
  3. Uncheck the Show welcome screen on startup checkbox.

    Tip: You can also switch this setting off from the welcome screen itself, by unchecking the Show this dialog at startup checkbox.

  4. The next time you start the QSP, you will not see the welcome screen.
  5. To turn the feature on again, check the Show welcome screen on startup checkbox in the application settings.
