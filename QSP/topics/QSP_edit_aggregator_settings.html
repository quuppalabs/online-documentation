Edit Aggregator Settings

The Quuppa Data Aggregator (QDA) is a web application that collects and aggregates data from multiple Quuppa Positioning Engines (QPE) and exposes the aggregated data using the Aggregator API. The QDA settings can be edited in the QSP if you have enabled the QDA Options from the application settings. For more information on how to do this, please see the Enable QDA Options section.

Once the QDA Options are enabled, you can edit the QDA settings by following the steps below.

  1. Open the project in the QSP.
  2. In the object tree on the left, select the project file.

  3. In the panel on the right, navigate to the Aggregators tab.

    Note: The Aggregators tab is only available if you have enabled the QDA options in the application settings. If you cannot find the tab, please see the Enable QDA Options section.
  4. Edit the Aggregator settings as needed:
    • Add Aggregator: You can add a new Aggregator for the project by following the steps below.
      1. Click the Add Aggregator button.

      2. In the window that opens, add a name (Server ID) and IP address for the Aggregator and select which the Tag groups for which the data will be aggregated.

      3. Click the OK button.
    • Edit Aggregator: Edit the Aggregator settings as needed by following the steps below.
      1. Click the Edit Aggregator button.

      2. In the window that opens, edit the Aggregator settings as needed.
      3. Click the OK button.
    • Remove Aggregator:
      1. Click the Remove Aggregator button.

      2. The Aggregator will be removed.
