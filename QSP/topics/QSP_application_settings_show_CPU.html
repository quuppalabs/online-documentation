Show Memory and CPU Usage

In some cases, you may want to see memory and CPU load data as you edit the project file using the QSP tool. This feature is not enabled by default, but can be turned on from the Application Settings by following the steps below.
  1. Open the QSP.
  2. In the menu bar at the top, select the Settings menu and select Application Settings. The Application Settings window will open.
  3. Click the checkbox for Show memory and CPU usage (requires restart) to activate the feature.

  4. Restart the QSP to enable the feature. Once restarted, the memory and CPU data will be visible in the bottom right corner of the QSP.
