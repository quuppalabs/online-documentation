Application Settings

The QSP operates using certain pre-defined default settings. You can customise these QSP settings by editing the application settings.

For example:
  • Font size (both in the map view and the UI in general)
  • Unit system shown by the QSP (metric or imperial)
  • WGS-84 options
  • Legacy Focusing Locator setting
  • The frequency channel options
  • Coverage estimate visualisation options (default colours or high contrast colours)

In the following chapters, you can find information about the available settings, what they do and how to use them.
