Background Images

The project background image serves as a visual guide for the project deployment. It is typically a floor plan or blueprint of the deployment environment. You can add multiple background images to your project if you wish, or you can have multiple coordinate systems using the same background image.

Note:
The maximum project file size that can be submitted to the Quuppa Customer Portal is 16 MB. This includes the background images if you chose to submit them.

This section will guide you through adding, scaling and adjusting the background image settings in the QSP.
