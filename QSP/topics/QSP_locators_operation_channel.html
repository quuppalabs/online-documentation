Edit Operation Channel

The Operation Channel feature allows users to adjust the system according to their needs. In cases where only Quuppa Compatible Tags are tracked, then the proprietary channel is recommended to reduce the amount of traffic processed by the Locators. However, in cases where other Bluetooth devices are also being tracked, the Bluetooth operation channel will be needed. The channel can be changed in the QSP.

Follow these steps to change the Operation Channel.

Note: You can also use the Locator Table to change the operation channel. For more information about the Locator Table, please see the View & Edit Locator Settings Using Locator Table section.
  1. Open your project in the QSP
  2. In the QSP object tree on the left, select the Locator or Locators that you want to adjust operation channel for.
  3. In the object info panel on the right, change the operation channel from the drop-down menu. Note that the panel for editing a single Locator is a bit different to the one for editing multiple Locators.
    1. To adjust a single Locator, go to the Config tab in the object info panel and select the channel that you want from the drop-down menu.

    2. To adjust multiple Locators simultaneously, select the channel that you want from the drop-down menu. To apply your selection, make sure to click the Set channel button next to the drop-down menu.
