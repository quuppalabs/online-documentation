Configure Selected Tags

When you are ready to upload a new configuration for the tags, you have two choices: autoconfiguration or manual configuration. Configuring selected tags is handy when you only want the configuration to affect certain selected tags.

To configure selected tags, follow the steps below:

  1. Select the tags you want to configure by holding down Ctrl and choosing the selected tags by clicking with the left mouse button.

    Tip: You can also select or unselect all tags by opening the Selection menu and choosing Select all or Unselect all.
  2. If the tags are not yet assigned to a tag group, open the Selection menu and select Set tag group. Then select an appropriate group from the drop-down list and select OK.

  3. Selected tags are now assigned to the specified tag group and given a name.

  4. Open the Selection menu and select Configure Selected tags....
  5. In the window that opens, you can see the progress of the configuration process.

  6. The system will inform you when the configuration is ready.

  7. Close the window.
