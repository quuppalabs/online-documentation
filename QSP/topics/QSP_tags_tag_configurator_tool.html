Configure Tags

Tag configurator tool is used to upload a tag configuration (as defined with the tag configuration editor) to tags. During this process, tag firmware is updated to behave according to the new configuration. This section will walk you through using the tag configurator tool to configure tags.

In order to configure the tags, you will need to have access to the tags, a Focusing Locator and the QSP.
