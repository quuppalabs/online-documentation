Import or Export Project Objects

In cases where you would like to import or export project objects from one Quuppa project to another, you can do so by using QSP's import and export tools. To do so, follow the steps below.

Export Project Objects

  1. Open the project that you want to export project objects from in the QSP.
  2. Open the Project menu and select Export project object data.

  3. In the Export project object information window that opens, select which objects to include in the export and click on the Export to file button.

  4. Select where to save the exported files on your computer and click the Save button to export the files. The exported objects will be saved as a .quuppaobjectjson file.

Import Project Objects

  1. Open the project that you want to import the project objects to in the QSP.
  2. Open the Project menu and select Import project object data.

  3. Select the .quuppaobjectjson file that you want to import into the QSP and click the Open button.
  4. In the Import project object information window that opens, select the objects that you want to import and click on the Import button.
