Create Zone Groups

The Zone Groups feature enables the editing of multiple zones at the same time. This makes it easy and efficient to add zone properties for similar zones within the same project. For example, in a hospital, the location-based commands for all of the patient rooms on a certain floor might be identical. In such cases, it is much more effective to change the settings for all zones in a zone group at once, rather than editing the settings for each zone separately.

To create zone groups, follow the steps below.

  1. Open the project in the QSP.
  2. In the menu bar at the top, open the Planning menu and select Add Zone Group.

    Tip:
    Alternatively, you can right-click on the zone group object in the object tree and select Add Zone Group.
  3. The QSP will prompt you to give the Zone Group a name.

    Note:
    This name can be edited later if needed by selecting the Zone Group in the object tree on the left and editing the text in the Name box shown in panel on the right.
  4. Edit the zone group properties in the panel on the right as needed. For example, you can edit the colour, the name and notes for the Zone Group and the location-based commands for zones in this group. You can also create zones that do not report location data by selecting either All tags or Custom for the Do not report location setting.

  5. Add the relevant zones to the Zone Group by selecting them in the object tree on the left and changing the Zone Group setting in the panel on the right. This can either be done one by one or by selecting several zones at once and editing their Zone Group simultaneously.
