Install Q-Link Locators

Once you have a project plan, you can start installing your system accordingly. This section will provide you with instructions related to the installation of Q-Link Locators and checking connectivity in preparation for deploying the Locators.

  1. Physically Install the Locators, according to your project plan and taking the following into account.
    1. Hub Locators:
      1. Install the hub Locators in their fixed locations:
        • If used for positioning and connectivity (Option A), the hub Locators should be installed on the ceiling like normal Locators.
        • If only used for connectivity (Option B), the hub Locators can be installed more freely, as long as they are in fixed locations (moving the device may cause the connection to the satellites to be lost) with no obstructions between them and the satellite Locators that could prevent radio signals from passing between the devices. For example, a person standing directly between the hub Locator and the satellite Locators could prevent the signal from passing through.
      2. Connect the hub Locators to the QPE as normal wired Locators.
    2. Satellite Locators:
      1. Install the satellite Locators so that their distance to the hub Locators do not exceed Quuppa's recommendations: 15 metres in confined or semi-confined environment and 40 metres in open environment. Also check there are no significant obstacles between the hub Locators and the satellite Locators that could block radio signals passing from one device to the other.
      2. Power the satellite Locators using USB cables and chargers (5 V, min. 500 mA).
        Warning:
        Don't connect the Locator with both PoE and USB power at the same time as this may damage the device. If your satellite Locator is still connected to the QPE using an Ethernet cable from the firmware upgrade steps above, first remove the Ethernet cable before connecting the USB power cable.
  2. Check Connectivity
    1. Once the hub Locators are connected, the QPE will automatically manage the connections between the hub Locators and the satellite Locators. The connection between the QPE and the satellite Locators is dependent on the hub Locator's connection to the QPE.
    2. Check the Locator connection status either in the Connection column of the QPE Web Console’s Locator Details table (a status of OK mean that you can proceed with the deployment) or by monitoring the Locator indicator lights:
      Sequence Sequence Description Indicator Light Mode
      Blinking blue Locator in deployment mode
      Static blue Locator in tracking mode
      Static red Locator powered, not connected
      Blinking red Locator connected to network

    For example: If a hub Locator is connected to the QPE (i.e. it's in deployment or tracking mode so the indicator light is blinking or static blue) but the satellite Locator isn't connected automatically (i.e. it's powered by not connected so the indicator light is showing a static red light), the QPE has been unable to connect to the satellite Locator via the hub Locator. Give the satellite a moment to try and establish a connection (up to 1 minute) and if no connection is made then try moving the satellite Locator closer to the hub Locator to establish a connection. Also check that there are no obstacles between the two Locators that could be blocking the connection.

    Note:
    If the satellite Locators drop the connection for some reason, it may take a couple of minutes for them to get back online and back into the Deployment Mode.
    Remember:
    The Q-Link module needs to be enabled for the subscription that you are using.
    Once all the Locators are connected to the QPE, you can proceed with identifying and focusing the Locators as you would in a normal Quuppa deployment using the Focusing Locator and the QSP. This process in described in the Deploy Quuppa Locators section.
