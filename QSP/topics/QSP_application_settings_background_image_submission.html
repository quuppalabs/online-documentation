Edit Background Image Submission Settings

The QSP application settings allow you to edit the default setting for whether the project's background image is submitted to the Quuppa Customer Portal (QCP) together with the project file. By default, the system is set to ask whether or not to submit the background image every time you submit the project. In the application settings you can change this setting to always include the background image or to always exclude the background image.
Important:
If you chose to exclude the background image, please make sure that you have a local copy of your project file saved to ensure that you still have access to the scaling parameters set for your project.
To change the setting, follow the steps below.
  1. Open the QSP.
  2. In the menu bar at the top, select the Settings menu and select Application Settings. The Application Settings window will open.
  3. Select the preferred setting (i.e. Ask, AlwaysInclude or AlwaysExclude) from the Include background image in project submission? drop down menu .

  4. The setting will be changed according to your selection.
