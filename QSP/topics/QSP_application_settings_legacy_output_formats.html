Enable Legacy Output Formats

By default, the QSP's API Data Formats & Output Targets editor does not show legacy output formats from older software versions that have been deprecated. However, if you need to access the legacy output formats for whatever reason, e.g. your existing project has not been updated to use the new formats yet, you can activate them through the Application Settings. To do so, follow the steps below:
  1. Open the QSP.
  2. In the menu bar at the top, select the Settings menu and select Application Settings. The Application Settings window will open.
  3. Click the checkbox for Legacy Output Formats to activate the feature.

  4. Close the Application Settings window. The legacy output formats will now be available in the API Data Formats & Output Targets editor.
