Useful Tips

Below you'll find some useful tips that will help you work more efficiently with the Positioning Engine Web Console.

Useful Icons

The table below covers some useful icons that are used in the Positioning Engine Web Console.

Icon Description
Hovering your mouse over the more information icon displays additional information and/or tips about using a specific feature.
Clicking on the add/edit configuration icon will open a window that enables you to create a new configuration for the item in question.

Useful Keyboard Shortcuts

The table below covers some of most common keyboard shortcuts that you can use when interacting with the Positioning Engine Web Console.

Shortcut Description
shift + f5 Reload current page, ignoring cashed content

API Documentation

The Quuppa API documentation can be accessed in multiple ways:

  • A link to the API documentation can be found in the links section at the bottom of the QPE Web Console's main page.
  • A PDF version of the API documentation can be downloaded from the Quuppa Customer Portal's Downloads section.
  • A PDF version of the API documentation is also available in the Quuppa software package.

Legacy Console

The current Positioning Engine Web Console was introduced with software release 7.0. However, if you need to access the previous version of the Web Console, you can do so via the Legacy Console link in the top right corner of the web console.

Navigation Tips

  • Main Page - You'll find various links and buttons that you can use to navigate to additional information about your project, e.g. the Locator and Tag Info tables, the API documentation and the License Details page.
  • Map View - If you want to return to the main view, you can either click the Quuppa logo in the top left corner or use your browsers navigation bar to return to the main page.
  • API Documentation - To navigate back to the main page from the API documentation, use your browser's navigation bar.
