Troubleshooting

This section will cover some of the basic troubleshooting tips for the QPE. If you have further questions, please contact Quuppa Support by emailing support@quuppa.com.

I cannot find a feature described in the documentation

Please check that you are using the latest version of the QPE software, available for download in the Quuppa Customer Portal's Downloads section. We are constantly working on improving the features and performance of the QPE and new features are made available in the new software releases. To make sure you have access to all available features, always update your QPE to the latest version. For best results, always update all of the Quuppa software tools at the same time to make sure the are running the same version. You can even enable notifications in the Quuppa Customer Portal to make sure that you do not miss any updates.

I have downloaded the latest software but my Positioning Engine Web Console still shows that it is running the previous version

Please make sure that you have followed all of the steps outlined in these instructions for updating the QPE software, including replacing the previous version of the qpe.war file with the new version of the qpe.war file.

I have made changes to my project in the QSP but can't see them in my deployment

Please make sure that you have saved the changes, resubmitted your project to the Quuppa Customer Portal and synced files for the QPE. The QPE gets its information via the Quuppa Customer Portal and so it's important to have the latest version of the project submitted to the Quuppa Customer Portal. Make sure that the QPE is definitely using the latest submitted project, by using the Web Console's do file sync button.

In the map view, I can see the tags moving but not the background image for the project

Please make sure that you have included the background image when submitting the project file to the Quuppa Customer Portal. Alternatively, you can also add the image file directly to the /bgimages folder in the QPE working directory (e.g. /opt/quuppa/bgimages/).

What if I want to build my own custom dashboard to use instead of the Positioning Engine Web Console?

You can use the Quuppa APIs to build a custom dashboard that better suits your use case requirements. For more information about the available APIs, please refer to out API documentation either via the QPE Web Console or the Quuppa Customer Portal's Downloads section.
