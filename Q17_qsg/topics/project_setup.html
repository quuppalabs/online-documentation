Setting up Generation Q Only Projects

Follow these steps to set up a project that only includes Generation Q Locators.
Note: You will need to have a valid subscription for all of the Generation Q Locators in this project.
  1. To create a new project or start using an existing project:
    1. Open the latest version of the QSP.
    2. Create a new project and select the appropriate default Locator type (e.g. if the majority of your Locators are Q17, select Q17) .
      Note: Alternatively, you can also open an existing project.
    3. Add new Locators to the project.

    4. Install the Locators.
    5. Position, identify and focus the Locators.
    6. Configure the tags.
    7. Save the project.
      Note: For more information about these steps, please refer to the Development Kit Quick Start Guide.
  2. To find the right key for the project in the QCP:
    1. In the dashboard, click on My Subscriptions to open the list of available subscriptions.
    2. Select the relevant subscription by clicking on the Subscription ID number to open the Subscription details page.
    3. Create a new project key by clicking the Create project key associated to this Subscription button.

      Note: It is good practice to add a note (e.g. the name of the project) to the project key.

      Note: Once the project key has been created, the details can be viewed in the Project Key Details page: Dashboard > Company name > Project Keys.
  3. To associate your project with the new project key:
    1. Return to the project in the QSP.
    2. Select the project folder in the object tree on the left.
    3. In the object info panel on the right, go to the General tab and click the Associate with a key button.

    4. Find the relevant project key in the panel that opens, select it and click the Associate with selected key button.
  4. Submit the project in the QSP by clicking the Submit project button in the object info panel on the right and close the QSP.

  5. To activate the project in the QPE Web Console:
    1. Start the QPE.
    2. Open the QPE Web Console.
    3. Click the Set New Key button to associate the project with the project key. The QPE will perform a file sync.

    4. Start the QPE in tracking mode.
      Note: It may take a moment for all of the Locators to show in the Locator table.
    Congratulations, your project is now up and running!
