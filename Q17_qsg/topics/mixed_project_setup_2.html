Setting up Mixed System Projects

Follow these steps to set up a project that includes both Generation Q and LD Generation Locators.
Note: Mixed system projects can include Generation Q Locators from multiple different subscriptions, but all included LD Generation Locators must be on one license. To run the project, you should use the project key that has been automatically generated for the license.
  1. To create a new project or start using an existing project:
    1. Open the latest version of the QSP.
    2. Create a new project and select the appropriate default Locator type (e.g. if the majority of your Locators are Q17, select Q17) .
      Note: Alternatively, you can also open an existing project.
    3. Add new Locators to the project.

    4. Install the Locators.
    5. Position, identify and focus the Locators.
    6. Configure the tags.
    7. Save the project.
      Note: For more information about these steps, please refer to the Development Kit Quick Start Guide.
  2. To find the right project key (i.e. the one linked to the LD Generation Locators) from in the QCP:
    1. In the dashboard, click on Licenses to open the list of available licenses.
    2. Select the relevant license by clicking on the Licence Key number to open the License details page.
    3. From the License details page, click the project key link to navigate to the Project Key Details page.

    4. Link the project key to the subscription that you want Generation Q Locators to be associated with by clicking the Change button next to the Linked with Subscription heading. This will allow any so far unassociated Locators used in the project to be automatically associated with the correct subscription.
      Note: Any Generation Q Locators already associated with subscriptions will remain on their original subscription. If all of your Locators are already associated with subscriptions, you can skip this step.

    6. In the Subscription list that opens, link the project key with the right subscription by clicking the relevant Link with this Subscription button in the Action column.

  3. To associate your project with the new project key:
    1. Return to the project in the QSP.
    2. Select the project folder in the object tree on the left.
    3. In the object info panel on the right, go to the General tab and click the Associate with a key button.

    4. Find the relevant project key in the panel that opens, select it and click the Associate with selected key button.
  4. Submit the project in the QSP by clicking the Submit project button in the object info panel on the right.

  5. To activate the project in the QPE Web Console:
    1. Start the QPE.
    2. Open the QPE Web Console.
    3. Click the Set New Key button to associate the project with the project key. The QPE will perform a file sync.

    4. Start the QPE in tracking mode.
      Note: It may take a moment for all of the Locators to show in the Locator table.
    Congratulations, your project is now up and running!
