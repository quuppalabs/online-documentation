Physical Installation

Once you have a project plan, you can start the physical installation of your system.
Tip: It is recommended that Locators should be aligned in a way that allows all Locators to point towards one of the cardinal points. This will make Locator alignment in the QSP much easier, as mentioned in Align Locators Using Rotation Angle.

Quuppa’s recommendation is to align all Locator LEDs in the same direction. For example, install all Locators so that their LEDs face one of the cardinal points.

  1. Install the Locators physically according to your plan. To install the Locators, please follow the mounting instructions that can be found on the Locator box.
    Note: For the best performance, install the Locators horizontally on the ceiling structures. For temporary installations, you can use e.g. zip ties or magnets to attach the Locators to the ceiling.
  2. Connect the Locators to the PoE switch using Ethernet cables. The Locator's indicator light will blink blue.
    Note: If the PoE switch is not connected to the Quuppa Controller, the Locator's indicator light will be static red. To fix this, turn on the PoE switch and restart the DHCP server by double-clicking the Restart DHCP Server icon.
