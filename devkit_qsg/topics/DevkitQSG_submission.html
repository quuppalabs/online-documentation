Project Submission

Almost there! To start using you project, associate the project with a project key and submit the project using the Quuppa Customer Portal and the QSP.

  1. Log into the Quuppa Customer Portal by double-clicking the Customer Portal icon.

  2. Create a new project key:
    1. In the Dashboard, click Subscriptions to open a list your subscriptions.

    2. Select the appropriate subscription for this project from the list by clicking on the Subscription ID. This will open the Subscription details page.
    3. Click the Create Project Key associated to this Subscription button to generate a new project key with a unique ID.

    4. The Project Key details page for the new project key will open. Here you can see the details of the project key. Copy the project key UUID to your clipboard.
      Tip: It's good practice to add a note (e.g. the name of the project) to the project key. The note will make it easier to identify the project key later on, without having to remember the full UUID number.

  3. Return to the QSP to associate the project with the new project key:
    1. In the menu bar at the top, select Project > Associate with a key.
    2. Log in using your Quuppa Customer Portal user name and password.
    3. Find the relevant project key in the panel that opens, select it and click the Associate with selected key button.
  4. Submit the project by selecting Project > Submit project in the menu bar at the top of the QSP.
  5. Save the project.
  6. Activate the project in the QPE Web Console:
    1. Start the Quuppa Positioning Engine by double-clicking the icon on the Quuppa Controller's desktop and enter your password when prompted.

    2. Open the QPE Web Console by double-clicking the icon on the desktop.

    3. Click the Set Key button to associate the project with the project key.
    4. Paste the project key that you copied to your clipboard in step 1.d. and click the submit button.
      Note: If you lost the project key along the way, not to worry. You can find it again in the QSP or through the Quuppa Customer Portal.
    5. The QPE will perform a file sync automatically.
    6. Start the QPE in tracking mode, if it did not start automatically.
      Note: It may take a moment for all of the Locators to show in the Locator table.

    To see your dots on the map, click the Map View button.

    Congratulations! You are now ready to start tracking.
