Руководство по установке локатора Q17 и информация по технике безопасности

Note: Неофициальный перевод документа с английского языка. В случае несоответствий между документами приоритетной должна считаться оригинальная версия документа на английском языке.

Quuppa Q17 (локатор Quuppa) совместим с системой Quuppa Intelligent Locating System (интеллектуальная система позиционирования Quuppa). Другими компонентами Quuppa Intelligent Locating System являются маяки Quuppa и программное обеспечение Quuppa Positioning Engine (модуль позиционирования Quuppa). Quuppa Intelligent Locating System позволяет находить и отслеживать любые объекты с маяками Quuppa в зоне, где установлены локаторы Quuppa. Локатор Quuppa Q17 предназначен для использования в помещениях. Для использования локатора Quuppa Q17 требуется подписка на него.

Перед установкой Quuppa Q17 внимательно прочитайте данное руководство. Кроме того, перед началом использования системы просмотрите обучающие видео Quuppa Tutorial и Quuppa Academy. Чтобы загрузить самые свежие материалы по своему изделию Quuppa, перейдите на портал клиентов Quuppa https://secure.quuppa.com/.

Безопасность пользователя

Ознакомьтесь c этими рекомендациями. Несоблюдение данных рекомендаций может быть опасным или противоречить законодательству. Чтобы получить дополнительную информацию, прочтите руководство по установке полностью. При возникновении каких-либо вопросов обратитесь по адресу support@quuppa.com.

ВНИМАНИЕ!

Все беспроводные устройства могут быть подвержены помехам, влияющим на их работу.

Установку, настройку и ремонт данного изделия должны выполнять только квалифицированные специалисты.
