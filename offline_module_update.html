Update a project using the Offline Module

The Offline Module is an optional module for Quuppa Locator subscriptions, which allows the Quuppa Positioning Engine (QPE) to run a project completely offline.

Note:
The Offline Module is an optional module and must be purchased and activated in advance. Without this feature enabled in advance, the QPE is programmed to stop tracking after 48 hours of being offline, as it must periodically verify licence details via the Quuppa Customer Portal (QCP).

Because an offline QPE does not automatically connect to or sync with the Quuppa Customer Portal (QCP), any updates to the project file or license files must be transferred manually. For example, if you renew your Quuppa subscription or make changes to your project file, you must update these files to the QPE.

This section will walk you through updating the project and license files for your offline QPE.

Update license files offline

When running the QPE in offline mode, the system will not sync with the QCP, and the Locators will run for as long as their subscriptions are valid. If a subscription is updated (for example, if you renew your Quuppa subscription), you must transfer the new subscription files manually to the offline QPE. To do so, follow the steps below.

Attention:
You must ensure that your Quuppa subscription is active and paid in full before attempting to update the files. Transferring files from an inactive or unpaid subscription will cause the QPE to stop tracking.
  1. Verify subscription status: Log into the QCP using an online device and check the Subscription Details page to confirm the subscription is active.
  2. Navigate to Project Keys: From the Dashboard, navigate to the Project Keys list.
  3. Open Project Key details: Select the relevant Project Key ID to open the Project Key Details page.
  4. Download the files: Download the license, signature and the sync files from QCP by clicking the Download files button on the Project Key Details page.
    Note:
    If you submitted the latest project file to the QCP (optional with the Offline Module), these downloaded files will also include the project file and the project signature.
  5. Stop Tomcat.
  6. Transfer files: Unzip all the downloaded files and transfer them to the offline QPE's project folder (defined in the QPE configuration as -Dproject.folder) using a USB stick.
    Note:
    You can check the QPE’s project folder from the Working Directory row in the QPE Web Console.
  7. Start Tomcat.
  8. Verify tracking mode: Check in the QPE Web Console to verify that the QPE is running in tracking mode and that the Project Sync Status is Disabled. You are now running the project offline.

Update project file

If you need to make changes to a project file for a project that you are running using the Offline Module, update the file on the QPE:

  1. Update project file: Update and save the project file as needed using the Quuppa Site Planner (QSP).
  2. Transfer file: Use a USB stick to transfer the file over to the offline QPE.
  3. Stop Tomcat.
  4. Override existing file: Save the project file in the quuppaprojectoverride folder in QPE's project folder so that it replaces the existing file in that folder.
    Important:
    Only one .quuppaproject file can be stored in this folder at a time.
  5. Start Tomcat.
  6. Verify tracking mode: Check in the QPE Web Console that QPE is running in tracking mode and Project Sync Status is Disabled. You are now running the project offline.

Update files using an internet connection

Another way to update the license files for your offline QPE is to temporarily connect the QPE to the internet so that you can access the QCP for long enough to download the latest files.

  1. Connect to the internet: Connect your offline QPE to the internet.
  2. Sync files: In the QPE Web Console, click Do File Sync.
  3. Verify tracking and subscriptions: Check that QPE is running in tracking mode and on the Locator Details page, Locator subscription details have been updated correctly.

  4. Disconnect from the internet: Disconnect the QPE from the internet.
