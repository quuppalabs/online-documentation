Quuppa Data Player User Manual

The Quuppa Data Player (QDP) is an optional module to the Quuppa Intelligent Locating System. It enables Real-time viewing, recording, replaying, and analysing data generated by the Quuppa system.

The QDP can be downloaded from Quuppa Customer Portal’s Downloads section. Separate applications for Windows, Linux and Mac are available, as well as sample projects and recorded data for ice hockey, basketball and a retail store.

Using the Data Player requires a valid Software License with QDP option enabled, both for recording and for replaying data.

Contact Quuppa to get access to the QDP: support@quuppa.com
