Getting Started

The Quuppa Intelligent Locating System™ is a powerful one-size-fits-all technology platform for a variety of location-based services and applications. The system is made up of both software and hardware components, including the Quuppa software applications Locators and tags.

This section will walk you through the first steps in getting to know the Quuppa system, including setting up your very first Quuppa Development Kit and getting to know the Quuppa reference project files.

Included in this section:

  • Quuppa Development Kit Quick Start Guide
  • Quuppa Reference Project Documentation

Important: Only Quuppa Approved tags or Quuppa Tested Devices should be used when tracking physical tags. Tag emulation should only be used when tracking smart devices and in those cases the Quuppa tag emulation library should be used.
