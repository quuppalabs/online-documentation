QC2 Quuppa Controller Product Specification

The QC2 Quuppa Controller is a preconfigured device for running the Quuppa Intelligent Locating System. The system allows you to locate and track any objects equipped with Quuppa Tags in an environment where Quuppa Location infrastructure, including our Locators, has been installed.

System

Processor Intel® Core™ i3-1115G4 Processor (6M Cache, up to 4.10 GHz)
Processor speed 4.10 GHz
Processor Cores 2
Integrated Graphics Yes
Graphics Dual HDMI 2.0b w/HDMI CEC, Dual DP 1.4a via Type C
Memory 8GB
Storage 250GB SSD
Operating System Ubuntu 20.04 LTS
DC Input Voltage Supported 12-20 VDC
TDP 15 W

I/O specification

Number of USB Ports 4
USB Configuration

Front: 2x USB 3.2

Rear: 2x USB 4 (type C), 1x USB 3.2, 1x USB 2.0

Internal: 1x USB 3.2 on m.2 22x42 (pins), 2x USB 2.0 (headers)
Integrated LAN Intel® Ethernet Controller i225-LM
Integrated Wireless Intel® Wi-Fi 6 AX201
Integrated Bluetooth Yes

Expansion options

PCI Express Revision Gen 4 (m.2 22x80 slot); Gen 3 (otherwise)
PCI Express Configurations

PCIe x4 Gen 4: M.2 22x80 (key M)

PCIe x1 Gen 3: M.2 22x42 (key B)

PCIe x1: M.2 22x30 (key E)
M.2 Card Slot (wireless) 22x30 (E)
M.2 Card Slot (storage) 22x80 NVMe (M); 22x42 SATA (B)

Mechanical

Chassis Dimensions 117 x 112 x 37 [mm] (LxWxH)
